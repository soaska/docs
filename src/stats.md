# Статистика сервера {#stats}

Статистика работы SOCKS5 прокси сервера в режиме реального времени.

<StatsWidget />

## О статистике {#about}

Данная страница показывает публичную статистику работы прокси-сервера:

- **Total Connections** - общее количество подключений с момента запуска сервера
- **Active Now** - количество активных соединений в данный момент
- **Total Traffic** - общий объем переданных данных через прокси
- **Uptime** - время работы сервера без перезагрузок
- **Geographic Distribution** - распределение подключений по странам

Статистика обновляется автоматически каждые 30 секунд.

## Speed Test {#speedtest}

Вы можете запустить тест скорости сервера, нажав кнопку "Run Speed Test" выше. Тест измеряет скорость загрузки, выгрузки и пинг до ближайшего сервера.

::: info Ограничение
Тест скорости можно запускать не чаще одного раза в 10 минут для предотвращения перегрузки сервера.
:::

## Приватность {#privacy}

Публичная статистика не содержит:
- IP-адреса пользователей
- Детальную информацию о подключениях
- Персональные данные

Вся статистика анонимизирована и показывает только общие метрики работы сервера.
