> Прокси работает только с доменами Telegram (telegram.org, t.me, telesco.pe, telegram.me, tg.dev).
> Подключение к другим ресурсам через этот прокси невозможно.

## Telegram Desktop {#telegram-desktop}

1. Откройте Telegram Desktop
2. Перейдите в **Настройки** → **Продвинутые настройки** → **Тип подключения**
3. Выберите **Использовать пользовательский прокси**
4. Нажмите **Добавить прокси**
5. Выберите тип **SOCKS5**
6. Заполните поля:
   - **Сервер**: {{ ADDRESS }}
   - **Порт**: {{ PORT }}
   - **Имя пользователя**: оставьте пустым
   - **Пароль**: оставьте пустым
7. Нажмите **Сохранить**
8. Убедитесь, что прокси активирован (галочка напротив)

## Telegram Web {#telegram-web}

Telegram Web автоматически использует системные настройки прокси вашего браузера.

Для настройки прокси в браузере:

### Chrome/Edge/Brave

1. Откройте **Настройки** → **Система** → **Открыть настройки прокси-сервера компьютера**
2. Следуйте инструкциям для вашей операционной системы

### Firefox

1. Откройте **Настройки** → **Основные** → **Параметры сети** → **Настроить**
2. Выберите **Ручная настройка прокси**
3. В поле **SOCKS Host** введите {{ ADDRESS }}
4. В поле **Port** введите {{ PORT }}
5. Выберите **SOCKS v5**
6. Нажмите **OK**

## Проверка подключения {#check-connection}

После настройки прокси:

1. Telegram должен подключиться автоматически
2. В настройках прокси вы увидите статус подключения
3. Если подключение не работает, проверьте:
   - Правильность введенного адреса и порта
   - Доступность интернета
   - Актуальность адреса прокси-сервера

## Альтернативные способы {#alternatives}

### Прокси-ссылка (t.me)

Вы можете поделиться настройками прокси через специальную ссылку:

```
https://t.me/socks?server={{ ADDRESS }}&port={{ PORT }}
```

При переходе по такой ссылке Telegram автоматически предложит добавить прокси.

### QR-код

Некоторые версии Telegram поддерживают добавление прокси через QR-код. Сгенерируйте QR-код с прокси-ссылкой и отсканируйте его в приложении.
